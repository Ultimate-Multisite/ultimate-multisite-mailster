Seamlessly integrate Mailster email marketing with Ultimate Multisite to automatically build your subscriber lists.

== Description ==

Transform your multisite network into a powerful email list-building machine with the Ultimate Multisite Mailster Integration addon. Automatically add new customers to your Mailster email lists when they sign up or make purchases, creating a seamless connection between your membership platform and email marketing strategy.

= Key Features =

**Automatic Subscriber Sync**
* Automatically add customers to Mailster lists when they sign up
* Choose to sync immediately on signup or wait for payment confirmation
* Keep your email lists fresh and up-to-date without manual intervention

**Flexible List Management**
* Set global default lists for all new customers
* Override defaults with product-specific lists for targeted campaigns
* Easily manage list assignments with Mailster's familiar interface

**GDPR-Compliant Opt-In Options**
* Choose between automatic subscription or checkbox-based consent
* Enable double opt-in for confirmed subscriptions
* Show/hide opt-in checkbox based on your compliance requirements

**Smart Data Mapping**
* Automatically sync customer names and contact information
* Map billing addresses to subscriber fields
* Track signup dates for targeted campaigns

**Product-Level Control**
* Assign different lists to different products
* Create targeted campaigns based on what customers purchase
* Disable Mailster for specific products when needed

**Professional Experience**
* Native Mailster UI integration for list selection
* Detailed logging for troubleshooting and verification
* Seamless updates of existing subscribers

= Perfect For =

* Membership site owners building email lists
* SaaS platforms wanting to nurture new customers
* Course creators segmenting students by enrollment
* Anyone running Ultimate Multisite with Mailster

= How It Works =

1. Install and activate the addon on your Ultimate Multisite network
2. Configure global default lists in Ultimate Multisite settings
3. Optionally override with product-specific lists
4. Choose your opt-in mode and subscription timing
5. Watch your Mailster lists grow automatically!

== Installation ==

1. Upload the plugin files to your Ultimate Multisite addons directory
2. Ensure Mailster is installed and active on your main site
3. Activate the addon through the Ultimate Multisite addons page
4. Navigate to Settings > Mailster Integration to configure
5. Select your default lists and configure opt-in preferences

== Frequently Asked Questions ==

= Does this addon work without Ultimate Multisite? =

No, this addon requires Ultimate Multisite to function. It's designed specifically to integrate Mailster with Ultimate Multisite's customer and product system.

= Do I need Mailster installed on all sites? =

No, Mailster only needs to be installed and activated on your main network site. The addon will automatically connect to it from all subsites.

= Can I assign different lists to different products? =

Yes! You can set global default lists, then override them on a per-product basis. This allows you to create highly targeted email campaigns based on what customers purchase.

= What happens if a customer already exists in Mailster? =

By default, the addon will update existing subscribers and add them to new lists. You can disable this in settings if you prefer to skip existing subscribers.

= Is this GDPR compliant? =

Yes! The addon includes checkbox-based opt-in mode where customers must explicitly consent to receive emails. This meets GDPR requirements when combined with proper privacy policies.

= Can I use double opt-in? =

Absolutely! Enable the double opt-in setting and Mailster will send confirmation emails before activating subscriptions.
